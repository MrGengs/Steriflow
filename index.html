How SteriFlow Works

Automated sterilization from detection to completion in 6 smart steps

01

System Initialization

ESP32 activates all components: MQ3, MQ6, MQ8 gas sensors, DHT11, relay module, UV-C lamp and fan ready to operate.

02

Contamination Detection

VOC sensor analyzes air contamination. If the threshold is exceeded, the system is automatically activated.

03

Sterilization Process

UV-C lamp actively kills pathogenic microorganisms directly, along with 70% food-grade ethanol spray through the system nozzle.

04

Environment Monitoring

DHT11 sensor continuously monitors room temperature and humidity to ensure optimal conditions during the sterilization process.

05

Real-time Monitoring

Web dashboard displays real-time sensor data from Firebase Realtime Database. Monitor gas levels, temperature, humidity, and relay status from anywhere.

06

Process Complete

System automatically turns off all actuators after sterilization is complete. Logs are saved to the dashboard and the green indicator shows clean.
